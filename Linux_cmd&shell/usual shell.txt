【CENTOS7】
【文件操作】
ls -a 目录路径（显示所有文件（包括隐藏文件））
   -l 目录路径（显示文件详细信息）
   -h 目录路径（文件大小人性化显示，一般-lh）
   -d 目录路径（显示目录本身信息，一般-ld）
   -i 目录路径（显示各文件id号）
ll 目录路径（相当于ls -l）

mkdir 目录路径（创建目录）
      -p 目录路径（递归创建目录）

touch 文件路径（创建文件）
（若创建的文件名带空格，需用""括起来，如touch "my study"）

rm 文件路径                   （删除文件）
   -r 目录路径                （删除目录）   
   -f                         （删除时不确认）

cd 目录路径                   （切换到指定目录）
cd ..                         （返回到上一级目录）
pwd                           （显示当前目录）

mv 原文件或目录路径 目标文件或目录路径（剪切文件，在剪切同时可更名(在目标文件或目录路径后写上新命名)）

cp 原文件路径 目标文件路径    （复制文件）
   -r 原目录路径 目标目录路径 （复制目录）
   -p                         （复制同时保留原文件属性，如原文件最后修改时间等）
   -d                         （复制文件，若文件是链接文件则复制链接）

【文件浏览】
cat 文件路径                  （显示文件内容，一般用于显示小文件，tac倒着显示）
    -n 文件路径               （显示行号）

less 文件路径                 （分页显示文件内容，一般用于显示大文件）
（空格或f（翻页））
（回车   （换行））
（q      （退出））
（特殊：可支持用上箭头向上翻，可用/关键词进行搜索，按n，寻找下一个关键词）

行搜索：
grep "字符串" 文件路径           （提取文件中指定字串的行）
     -i "字符串" 文件路径        （不区分大小写） 
grep -v ^# 文件路径              （提取除了行首为#的行）

列搜索：
cut -f 列号 文件路径    （按列提起）
    -d "分割符" 文件路径（按分隔符提取）



【文件查找】
find 查找目录路径 -iname *文件名*  （在该目录中查找包含有该名字的文件）

locate 文件名    （文件资料库中查找包含文件名的文件，不能查找/tmp目录）
       -i 文件名 （不区分大小写查找）
updatedb         （手动更新资料库）

whereis 命令 （查找命令的路径及相关文件信息）
which 命令   （查找命令的路径及别名）
whatis 命令  （查看命令的简短信息）

man 命令或配置文件名（获取帮助信息，以1.结尾是命令帮助，以5.结尾是文件帮助，只能文件名）
    1 命令          （获取命令帮助信息）
    5 配置文件名    （获取配置文件帮助信息）


【压缩命令】（所有解压都是解压到当前路径下，使用前要先cd到要解压到的目录）
gzip 文件名  （将文件压缩成.gz格式，不保留原文件）
gunzip 文件名（解压，也可以用gzip -d）

zip 压缩后的文件名 文件名   （压缩文件.zip，压缩会保留原文件）
    -r 压缩后的目录名 目录名（压缩目录）
unzip 文件名                （解压.zip）

bzip2 文件名           （压缩.bz2）
      -k 文件名        （压缩且保存原文件）
bunzip2 文件名         （解压.bz2）
        -k 文件名      （解压且保留原文件）

tar -cvf （打包成.tar，压缩后的包写在前，要压缩的文件写在后）
    -j   （压缩成.bz2）
    -z    (压缩成.gz)
tar -xvf （解包.tar）
    -j   （解压.bz2）
    -z    (解压.gz)

【系统管理】
shutdown -h now（现在关机，now可以换成-> 小时:分钟，定时在几时几分关机）
         -r now（重启）

ping ip地址    （测试网络通断以及网络延时）
     -c ip地址 （指定发送次数）
（ctrl+c退出）

ifconfig       		（查看ip地址信息）
ifconfig 网卡名 新ip地址（设置暂时新的ip地址）


【安装包】（依赖性以.so结尾的包是模块，在www.rpmfind.net查询安装）
rpm包手动安装：
rpm -ivh 全包名（安装）
    -Uvh 全包名（升级）
    -e 包名    （卸载）
    -q 包名    （查询是否安装）
    -qa        （查询所有已安装包）
    -ql 包名   （查询已安装包中的文件和安装位置）

rpm包yum安装：
yum -y install 包名 （安装）
yum -y update  包名 （升级）
yum -y remove  包名 （卸载，慎用，可能卸载掉其他包与此包公用的依赖包）


【用户权限】（不可赋予普通用户vim（root）权限！！）
（掩码：r=4，w=2，x=1）
umask -S（查看新建默认的权限，新建文件时系统会自动去除x权限！！）

chmod u/g/o/a +/-/= r/w/x 文件路径 （给所有者/所属组/其他人/所有人，增加/减少/不增减，读/写/执行/的权限，多个授权时用逗号分割）
      -R                           （递归修改）

（如，chmod u+x,g+w /tmp/test，给此文件所有者增加可执行权限，所属组增加可写权限）
chmod 要修改的rwx掩码 文件路径     （改变某个文件的权限）
      -R                           （递归修改）

useradd 用户名（创建新用户）
passwd  用户名（设置用户密码）
userdel -r 用户名（删除用户及家目录，强制删除-f）

gpasswd -a 用户名 组名（加入组）
        -d 用户名 组名（踢出组）

su 用户名（只切换身份）
su - 用户名（切换为此用户环境变量也切换）
（可以env指令查看当前USER是谁来确认环境是否全部切换）
exit（退出当前用户）

sudo -l（查看被赋予了哪些root权限）

如何修改：/etc/sudoers执行时加sudo
用户名	ALL=要赋予的命令路径  （赋予部分权限）
用户名	ALL=!要赋予的命令路径 （赋予除了此命令的所有权限,禁用多个权限时，用逗号隔开）
用户名	ALL=(ALL)	ALL   （赋予所有权限）

【挂载】
df -h         （查看分区基本信息）
df -a 挂载目录（显示所有文件，包括特殊文件系统信息）（占用空间包含未释放的空间）
   -h 挂载目录（人性化显示）
   -T 挂载目录（显示文件系统类型）

du -sh 文件名 （自身文件大小，纯粹的占用空间）

fdisk -l      （查看磁盘设备文件名）
mount -l      （查看已经挂载的设备）
mount (-t 文件系统) 设备文件名 挂载点目录    （挂载外部设备，u盘等）
（括号内可省略）
如：mount (-t iso9660) /dev/sr0 /mnt/cdrom 
umount (-t 文件系统) 设备文件名或者挂载点目录（解挂）

mkfs -t ext4 设备文件名（格式化此设备为ext4系统）
partprobe              （磁盘配置完成后，执行一次此命令来重新读取分区表信息）


【脚本】
标准输出重定向：
命令 &>文件路径              （将一切结果覆盖方式保存）
命令 &>>文件路径             （将一切结果追加方式保存）
命令 >>文件1路径 2>>文件2路径（将命令正确结果保存于文件1，错误结果保存于文件2，追加方式）
命令 &>/dev/null             （将执行结果丢入回收站）
